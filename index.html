Activer le suivi en direct

Colle l'URL de ton relais Cloudflare. Scores, classements, buteurs et passeurs se mettront alors à jour automatiquement toutes les 60 s, sans rien refaire.

Comment ça marche :
Le relais est un mini-service gratuit (Cloudflare Worker) qui détient ta clé API-Football et la cache du navigateur. Ta page l'interroge, il renvoie les données. C'est ce qui rend le live possible depuis une page statique.
🔒 Ta clé API n'est jamais dans ce fichier : elle reste dans le relais. Seule l'URL du relais est stockée dans ton navigateur.
